# 🎮 **Starknet-Arcade (Arcadino)**

Arcadino is a fully on-chain GameFi project built on the StarkNet ecosystem.  
100% Fair, fully on-chain, gasless, and provably transparent - powered by StarkNet VRF and session keys.
